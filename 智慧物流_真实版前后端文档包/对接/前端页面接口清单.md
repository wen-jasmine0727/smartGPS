# 前端页面接口清单

## 1. 登录页

| 动作 | 接口 | 说明 |
|---|---|---|
| 登录 | `POST /auth/login` | 保存 token、user、role |
| 获取当前用户 | `GET /users/me` | 刷新页面后恢复用户信息 |

---

## 2. 货物追踪页：前端 1

| 区域 | 接口/协议 | 用途 |
|---|---|---|
| 货物详情卡片 | `GET /cargo/{cargoId}` | 货物、司机、起终点、进度 |
| 实时位置 | `GET /cargo/{cargoId}/position` | 首次加载或兜底轮询 |
| 地图实时刷新 | `WebSocket vehicle.position` | 车辆位置推送 |
| 轨迹折线 | `GET /cargo/{cargoId}/trajectory` | 地图轨迹线 |
| ETA | `GET /cargo/{cargoId}/eta` | 预计到达、剩余距离、趋势 |
| 时间线 | `GET /cargo/{cargoId}/timeline` | 装货、离仓、途经、告警、预计到达 |
| 告警提示 | `WebSocket alert.triggered` | 页面顶部或侧栏提示 |

---

## 3. 车辆调度总览页：前端 1

| 区域 | 接口/协议 | 用途 |
|---|---|---|
| 车辆统计 | `GET /vehicles` | 统计在线/移动/离线 |
| 车辆地图 | `GET /vehicles` + `WebSocket vehicle.position` | 所有车辆位置 |
| 搜索筛选 | `GET /vehicles?keyword=&status=` | 筛选列表 |
| 车辆详情 | `GET /vehicles/{id}` | 点击车辆详情 |
| 下发指令 | `POST /vehicles/{plate}/command` | 调度操作 |
| 指令状态 | `GET /vehicles/{plate}/command/{commandId}` + `WebSocket command.ack` | 回执展示 |

---

## 4. 告警中心：前端 2

| 区域 | 接口/协议 | 用途 |
|---|---|---|
| 告警统计 | `GET /alerts/stats` | 顶部统计卡片 |
| 告警列表 | `GET /alerts` | 表格 |
| 告警筛选 | `GET /alerts?severity=&type=&status=` | chip 筛选 |
| 告警详情 | `GET /alerts/{alertId}` | 右侧详情 |
| 确认告警 | `POST /alerts/{alertId}/acknowledge` | 状态流转 |
| 关闭告警 | `POST /alerts/{alertId}/resolve` | 状态流转 |
| 新告警推送 | `WebSocket alert.triggered` | 实时提醒 |

---

## 5. 车辆货物管理页：前端 2

| 区域 | 接口 | 用途 |
|---|---|---|
| 车辆列表 | `GET /vehicles` | 展示所有车辆 |
| 新增车辆 | `POST /vehicles` | 表单提交 |
| 修改车辆 | `PUT /vehicles/{id}` | 编辑信息 |
| 删除车辆 | `DELETE /vehicles/{id}` | 删除/解绑 |
| 货物列表 | `GET /cargo` | 可绑定货物 |
| 新建货物 | `POST /cargo` | 新增运单 |
| 绑定 | `POST /cargo/bind` | 绑定货物车辆 |
| 解绑 | `POST /cargo/unbind` | 解除绑定 |
| 状态上报 | `PUT /cargo/{cargoId}/status` | 已装货/运输中/已送达 |
| 设备在线 | `GET /devices/status` | 在线/离线状态 |

---

## 6. 智能问答：前端 2

| 动作 | 接口 | 用途 |
|---|---|---|
| 推荐问题 | `GET /assistant/suggestions` | 快捷问题 |
| 发送问题 | `POST /assistant/chat` | 问答 |
| 会话历史 | `GET /assistant/sessions/{sessionId}/messages` | 聊天历史 |
| 上传知识库 | `POST /knowledge/documents` | 可选管理功能 |
| 文档索引 | `POST /knowledge/documents/{documentId}/index` | 可选管理功能 |
